# Discord Message Deletion Tool (Python)

This project is a **Python-based tool** designed for users who want to **delete their own messages**
in Discord direct messages with **selected users**.

The tool only removes messages **sent by the authenticated Discord account**.
It does **not** delete messages sent by the other party.
